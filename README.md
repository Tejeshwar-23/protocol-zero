# Protocol Zero: Tactical Strategy Simulation

A turn-based strategy game set in a dystopian cyberpunk universe.

## Features
- **Tactical Grid Combat**: Manuver units on an 8x8 grid with fog of war.
- **PvP & PvE Modes**: Challenge the AI or battle friends via code-based lobbies.
- **Tutorial Simulation**: Interactive training module to learn game mechanics.
- **Cyberpunk Aesthetic**: Immersive CRT visuals, dark UI, and atmospheric sound design.

## Setup
1. Install dependencies: `npm install`
2. Start the server: `npm start`
3. Access the simulation: `http://localhost:3000`

## Technologies
- Node.js & Express
- SQLite (Leaderboard & User Data)
- HTML5 / CSS3 (No Frameworks)
- Socket.io / Polling (Multiplayer Sync)
